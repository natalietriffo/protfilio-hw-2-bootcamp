Projects

Project One: Chowhounds User Story: Any person with specific wants and needs for their restaurant experience can use our web application to find the perfect restaurant for them. The user will be asked for a zip code, followed by a series of questions. Based on the user's response to the zip code and the questions, a list of restaurants will be provided. The list also provides the user with a yelp ranking, approximate travel time (in minutes), and the option to “favorite” the restaurant.
